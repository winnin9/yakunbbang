> 버전: v1.0

---

## 1. 개요

**야근빵**은 야근 시간을 기록하고, 목표 대비 초과율을 빵 굽기 메타포로 시각화하는 앱인토스 미니앱이다.

---

## 2. 문제 정의

- 야근을 생각 없이 하면 개인 시간이 줄어드는 것을 자각하기 어렵다.
- 목표 종료 시간 대비 얼마나 늦어졌는지 수치로 볼 수 있으면 스스로 인식하는 데 도움이 된다.
- 친구들과 서로 야근 현황을 비교하고 싶은 니즈가 있으나, 공유할 수 있는 수단이 없다.

---

## 3. 목표

| 목표 | 설명 |
|------|------|
| 야근 자각 | 목표 대비 초과율을 수치로 보여줘 야근 습관을 인식하게 한다 |
| 월간 회고 | 한 달치 야근 패턴을 제빵사 등급으로 요약 제공 |
| 소셜 공유 | 월간 통계 카드를 SNS에 공유할 수 있게 해 바이럴 유도 |

---

## 4. 타겟 사용자

- 토스 앱 사용자
- 정기적으로 야근을 하며 업무 시간 관리에 관심 있는 직장인

---

## 5. 플랫폼

- **앱인토스 미니앱** (WebView 또는 React Native)
- **로그인:** 토스 로그인 자동 지원 (별도 구현 불필요)
- **데이터 저장:** 토스 Storage API (로컬 저장, 서버 불필요)
- **공유:** 토스 공유 API 활용

---

## 6. 핵심 개념

### 빵 굽기 메타포

| 개념 | 설명 |
|------|------|
| 빵 굽기 시작 | 야근 시작 + 목표 종료 시간 설정 |
| 오븐 | 야근 세션 저장소 |
| 빵 완성 | 야근 종료 |
| 빵 상태 | 초과율에 따라 결정되는 야근 결과 |
| 제빵사 등급 | 월간 빵 품질 기반 종합 평가 |

### 초과율 계산 공식

```
초과율(%) = (실제 종료 시간 - 목표 종료 시간) ÷ (목표 종료 시간 - 시작 시간) × 100
```

예시: 시작 20:00 / 목표 22:00 / 실제 종료 23:00 → 초과율 50%

## 7. 기능 상세 스펙 (MVP)

### 7-1. 야근 시작

- 오늘의 목표 종료 시간 입력 (시간 선택 UI)
- "빵 굽기 시작" 버튼 탭 → 야근 세션 시작
- 시작 시간은 버튼 탭 시각으로 자동 기록

### 7-2. 야근 종료

- "빵 완성" 버튼 탭 → 야근 세션 종료
- 종료 시각 자동 기록
- 초과율 계산 후 빵 상태 판정
- 결과 화면 표시 (빵 이미지 + 초과율 + 한마디)

### 7-3. 빵 판정 (7단계)

| 초과율 | 빵 상태 | 이모지 | 설명 |
|--------|---------|--------|------|
| 0% | 완성된 빵 | 🍞 | 목표 달성 |
| 1~20% | 노릇노릇한 빵 | 🟡 | 살짝 늦었어요 |
| 21~40% | 살짝 탄 빵 | 🟠 | 조금 타고 있어요 |
| 41~60% | 많이 탄 빵 | 🟤 | 많이 탔네요 |
| 61~80% | 심하게 탄 빵 | ⬛ | 거의 숯이에요 |
| 81~100% | 새까맣게 탄 빵 | 💀 | 완전히 탔어요 |
| 100% 초과 | 재가 된 빵 | ☠️ | 목표의 2배 이상 야근 |

### 7-4. 오븐 (기록 보관함)

- 세션별 저장 데이터: 날짜 / 시작 시간 / 목표 종료 시간 / 실제 종료 시간 / 초과율 / 빵 상태
- 날짜 기준 목록으로 표시
- 월별 탭으로 전환 가능

### 7-5. 월간 리포트

- 매월 1일 기준으로 이전 달 자동 집계
- 표시 항목:
  - 총 야근 횟수
  - 예쁜 빵 수 (완성된 빵 + 노릇노릇한 빵)
  - 탄 빵 수 (살짝 탄 이상)
  - 제빵사 등급

### 7-6. 제빵사 등급

> 기준: 이달 전체 세션 중 "좋은 빵(0~20% 초과)" 비율

| 좋은 빵 비율 | 등급 | 코멘트 |
|-------------|------|--------|
| 90% 이상 | 👨‍🍳 마스터 제빵사 | 야근도 내 손 안에 |
| 70~89% | 🧑‍🍳 실력파 제빵사 | 꽤 잘 구웠어요 |
| 50~69% | 🍞 성실한 제빵사 | 분전하고 있어요 |
| 30~49% | 🟠 탄내나는 제빵사 | 오븐 온도 점검 필요 |
| 29% 이하 | ☠️ 숯 전문가 | 이달 오븐이 너무 뜨거웠어요 |

> ⚠️ 이달 세션이 3회 미만이면 등급 산정 불가 → "아직 데이터가 부족해요" 표시

### 7-7. SNS 공유 카드

- 월간 리포트 기반 이미지 카드 생성
- 카드 포함 항목:
  - 야근빵 로고
  - 제빵사 등급 + 등급명
  - 예쁜 빵 수 / 탄 빵 수
  - 월/연도
- 토스 공유 API로 외부 SNS 공유
